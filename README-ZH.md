## Genius-Android是什么?

![icon](https://raw.githubusercontent.com/qiujuer/Genius-Android/gh-pages/assets/images/logo.png)

**Genius-Android** 是 **Android** 中一些常用的的方法集合, **Genius** 提供6个基本板块：

* `app` (**Ui**)
* `animation` （**动画**）
* `drawable` （**Material Draw**）
* `widget` （**Material控件**）
* `command` （**命令行**）
* `net tool` （**Ping、Dns...**）
* `util` （**常用方法,类**）
